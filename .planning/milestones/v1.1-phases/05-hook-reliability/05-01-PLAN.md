<objective>
Fix the root cause of missing project-scoped memories (DIAG-02) and harden graphiti-helper.py against hangs.

Purpose: The Graphiti MCP server currently overrides all group_id values to "global" because of a server-level GRAPHITI_GROUP_ID environment variable in docker-compose.yml. Removing this line lets the server respect per-request group_id values, enabling project-scoped memory isolation. Additionally, the MCPClient in graphiti-helper.py has no internal timeout, meaning a slow/hung server blocks hooks indefinitely until the hook-level timeout kills the process. Adding a 5s timeout makes hooks fail fast with a clear error.

Output: Fixed docker-compose.yml, hardened graphiti-helper.py, verified scope isolation via diagnose.py
</objective>
