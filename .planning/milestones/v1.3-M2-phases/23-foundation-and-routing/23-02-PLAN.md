    Pre-compiled regex PATTERNS object (use global flag):
    ```javascript
    const PATTERNS = {
      filePaths: /(?:^|\s|['"`(])([.~]?\/[\w./-]+\.\w{1,10})(?:\s|['"`)]|$)/g,
      functionNames: /\b([a-z][a-zA-Z0-9]*(?:_[a-zA-Z0-9]+)*)\s*\(/g,
      classNames: /\b([A-Z][a-zA-Z0-9]+(?:[A-Z][a-zA-Z0-9]*)*)\b/g,

      technicalTerms: /\b(hook|handler|dispatch|activation|sublimation|injection|config|state|session|episode|entity|node|edge|schema|migration|deployment|subagent)\b/gi,
      camelCase: /\b([a-z]+(?:[A-Z][a-z0-9]*)+)\b/g,
      snakeCase: /\b([a-z]+(?:_[a-z0-9]+){2,})\b/g,
    };
    ```

    `extractEntities(text, options = {})`:
    - Returns `Array<{ name, type, count, positions }>`
    - Uses `Map` keyed by normalized (lowercased) name to deduplicate
    - CRITICAL: Reset `pattern.lastIndex = 0` before each exec loop (Pitfall 6 from research)
    - Returns `Array.from(entities.values())`

    **Section 2: Domain Frame Classification (IV-10)**

    FRAME_KEYWORDS object with 4 keyword arrays (engineering, debugging, architecture, social). General is default -- no keywords.

    `classifyDomainFrame(text, options = {})`:
    - Lowercase input, count keyword hits per frame
    - Sort by score descending
    - Top frame = highest score (or 'general' if all scores 0)
    - confidence = topScore / totalKeywords (or 0.5 if no matches)
    - active_frames = all frames with score > 0 (or ['general'] if none)
    - Returns `{ current_frame, frame_confidence, active_frames }`

    **Section 3: Activation Map Operations (IV-03)**

    `updateActivation(activationMap, newEntities, options = {})`:
    - For each entity in newEntities: if already in map, increment convergence_count and refresh last_activated; if not, add with level 1.0 and sources ['direct_mention']
    - Returns updated map (shallow copy)

    `propagateActivation(anchorEntities, graphData, options = {})`:
    - Parameters: hops (default 1), decayFactor (default 0.5), minThreshold (default 0.3), convergenceBonus (default 1.5), domainFrameBonus (default 1.3), domainFrame (default null)
    - Initialize map from anchor entities (level 1.0)
    - BFS propagation: for each hop, for each entity above threshold, look up neighbors in graphData[id] (array of {id, domain?}), propagate with decay, apply domain bonus if match, apply convergence bonus if convergence_count >= 2
    - Clamp all levels to max 1.0 via `Math.min(1.0, ...)`
    - Returns activation map object

    `decayAll(activationMap, options = {})`:
    - Parameters: decayRate (default 0.1 per minute), now (default new Date())
    - For each entry: compute minutesElapsed, decayFactor = Math.exp(-decayRate * minutesElapsed), newLevel = level * decayFactor
    - Prune entries where newLevel < 0.01
    - Returns new map (does not mutate input)

    `checkThresholdCrossings(activationMap, threshold, options = {})`:
    - Returns array of `{ id, level, crossed: true }` for entries above threshold

    **Section 4: Scoring (IV-04, IV-12)**

    `computeSublimationScore(entity, predictions, currentContext, cognitiveLoad, options = {})`:
    - Formula: `activationLevel * surpriseFactor * relevanceRatio * (1 - loadPenalty) * confidenceWeight`
    - Uses options overrides or falls back to computeSurprise/computeRelevance
    - Default confidenceWeight = 0.8
    - Default loadPenalty = cognitiveLoad if number, else 0.3

    `computeSurprise(entity, predictions)`:
    - No predictions or no expected_topic: return 0.5
    - Entity name appears in expected_topic (case-insensitive): return 0.2
    - Otherwise: return 0.8

    `computeRelevance(entity, currentContext)`:
    - No context: return 0.5
    - Entity name appears in context (case-insensitive): return 0.9
    - Otherwise: return 0.3

    **Section 5: Operational Monitoring (OPS-MON-01, OPS-MON-02)**

    `checkSpawnBudget(state, config)`:
    - cap from `(config.reverie && config.reverie.operational && config.reverie.operational.subagent_daily_cap) || 20`
    - today = `new Date().toISOString().slice(0, 10)`
    - tracker from `state.processing.spawn_tracker || { date: today, count: 0, rate_limited: false }`
    - Reset counter if tracker.date !== today
    - Returns `{ allowed: tracker.count < cap && !tracker.rate_limited, remaining: Math.max(0, cap - tracker.count), rate_limited: tracker.rate_limited, tracker }`

    `recordSpawn(state)`:
    - Initialize spawn_tracker if missing, increment count, return state

    `setRateLimited(state, limited)`:
    - Initialize spawn_tracker if missing, set rate_limited = limited, return state
