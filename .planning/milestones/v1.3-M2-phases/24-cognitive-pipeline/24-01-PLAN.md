       - signals object has: `explicitRecall` (bool), `rateLimited` (bool), `semanticShiftScore` (float), `entityConfidence` (float), `needsInjection` (bool), `predictionsMatch` (bool)
       - options: `shiftThreshold` (default 0.4), `confidenceThreshold` (default 0.7)
       - Priority order: predictionsMatch (D-12) -> skip, explicitRecall (D-11) -> deliberation, rateLimited (PATH-06) -> hot, semanticShift >= threshold (IV-09) -> deliberation, entityConfidence < threshold -> deliberation, !needsInjection -> skip, default -> hot

    2. `detectSemanticShift(currentEntities, previousEntities, options = {})` -- Per D-10, IV-09:
       - Takes arrays of entity objects with `.name` property
       - Computes Jaccard overlap: |intersection| / |union| using lowercased names
       - Returns `{ shifted: boolean, overlapScore: number }`
       - Default `shiftThreshold` option: 0.3

    3. `detectExplicitRecall(text, options = {})` -- Per D-11, IV-11:
       - Patterns: `/\bdo you (?:remember|recall|know)\b/i`, `/\bwhat do you (?:know|remember) about\b/i`, `/\bhave (?:we|you|I) (?:discussed|talked about|mentioned)\b/i`, `/\bremember when\b/i`, `/\brecall (?:the|our|my)\b/i`
       - Returns boolean

    4. `estimateTokens(text)` -- Per IV-05:
       - Returns `Math.ceil(text.length / 4)` (rough heuristic: ~4 chars per token)

    5. `truncateToTokenLimit(text, limit)` -- Per IV-05:
       - If estimated tokens <= limit, return text unchanged
       - Otherwise truncate to `limit * 4` chars, find last `. ` for sentence boundary
       - If no sentence boundary found, find last space for word boundary

    6. `formatHotPathInjection(crossedEntities, state, options = {})` -- Per D-01, D-02, D-03:
       - Template-based formatting (no LLM call)
       - Contextual narrative tone: "When you worked on {entity}, you {context}."
       - Adversarial framing: wraps facts with "from your experience" / "as you described it" qualifiers per D-03
       - Accepts `tokenLimit` option (default 150 per mid-session), truncates if needed
       - Returns formatted string or null if no entities qualify

    7. `adjustThreshold(currentThreshold, injectionHistory, options = {})` -- Per D-09:
       - `recentWindow` option (default 10)
       - Needs at least 3 entries to adjust
       - Acknowledged rate > 0.7: lower by 0.02 (min 0.3)
       - Acknowledged rate < 0.3: raise by 0.02 (max 0.9)
       - Otherwise: no change
