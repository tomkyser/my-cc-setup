<objective>
Create the inner-voice.cjs pipeline orchestrator -- the central module that implements per-hook cognitive processing pipelines. Each pipeline function loads relevant data, processes the event through a sequence of deterministic steps (entity extraction, activation updates, path selection, threshold evaluation, injection formatting), and returns an injection result or null plus updated state. Also implements the state bridge pattern for deliberation result write/consume with correlation ID and TTL.

Purpose: This is the brain of Reverie. Every handler (Plan 04) delegates to this module for its cognitive processing. It wires together activation.cjs (Phase 23), dual-path.cjs (Plan 01), and curation.cjs (Plan 02) into coherent pipelines.
Output: subsystems/reverie/inner-voice.cjs and its test suite

    **processUserPrompt(promptData, state, pendingResult, options = {}):**
    Returns `{ injection: string|null, updatedState: object, timings: object, pathSelected: string }`

    Steps with per-step timing via `performance.now()` (PATH-02):
    1. `statePrep`: Deep-copy state, load config (from options.config or loadConfig())
    2. `entityExtraction`: Call `extractEntities(promptData.prompt)`, store as currentEntities
    3. `activationUpdate`: Call `updateActivation(state.activation_map, currentEntities)`, `decayAll()`, update `state.activation_map`
    4. `domainClassification`: Call `classifyDomainFrame(promptData.prompt)`, update `state.domain_frame`
    5. `semanticShift`: Call `detectSemanticShift(currentEntities, state._previous_entities || [])`, per D-10
    6. `recallCheck`: Call `detectExplicitRecall(promptData.prompt)`, per D-11
    7. `predictionCheck`: Compare currentEntities against `state.predictions.expected_topic`, per D-12: if match, return skip
    8. `sublimation`: For each activated entity, call `computeSublimationScore()`, call `checkThresholdCrossings()` with threshold from config or `DEFAULT_SUBLIMATION_THRESHOLD`
    9. `pathSelection`: Build signals object `{ explicitRecall, rateLimited: spawnBudget.rate_limited, semanticShiftScore: shift.overlapScore inverted, entityConfidence: domainFrame.frame_confidence, needsInjection: crossedEntities.length > 0, predictionsMatch }`. Call `selectPath(signals)`
    10. **400ms abort check** (PATH-02): If `performance.now() - start > HOT_PATH_ABORT_MS`, log abort via `logError('reverie-hot-path', ...)`, return `{ injection: null, updatedState: state, timings, aborted: true }`
    11. `formatting`: If path is 'hot' and crossedEntities exist: call `formatHotPathInjection(crossedEntities, state, { tokenLimit: TOKEN_LIMITS.mid_session })` or if pendingResult provided: use pendingResult.injection
    12. Update self_model: `state.self_model.attention_state` = domain frame description, update `state.self_model.confidence`
    13. Update injection_history if injection produced: push `{ timestamp, acknowledged: false, entities: crossedEntities.map(e => e.id) }`
    14. Adjust threshold: `state._sublimation_threshold = adjustThreshold(state._sublimation_threshold || DEFAULT_SUBLIMATION_THRESHOLD, state.injection_history)`
    15. Store `state._previous_entities = currentEntities` for next semantic shift comparison
    16. If path is 'deliberation': set `state.processing.deliberation_pending = true`, `state.processing.last_deliberation_id = crypto.randomUUID()`, `state.processing.deliberation_type = recallCheck ? 'explicit_recall' : 'semantic_shift'`, `state.processing.deliberation_queue = crossedEntities`, and if spawnBudget allowed: `recordSpawn(state)`. Log soft cap warning if `remaining < 10` per D-13
    17. Return `{ injection, updatedState: state, timings, pathSelected }`

    **processSessionStart(sessionData, state, options = {}):**
    Returns `{ briefing: string, updatedState: object }`
    1. Reset session-scoped fields: `state.session_id = crypto.randomUUID()`, `state.processing.deliberation_pending = false`
    2. Decay activation map: `state.activation_map = decayAll(state.activation_map)`
    3. Classify domain frame from sessionData context (if available)
    4. Per D-14: always deliberation. Set `state.processing.deliberation_pending = true`, `state.processing.last_deliberation_id = crypto.randomUUID()`, `state.processing.deliberation_type = 'session_briefing'`, `recordSpawn(state)`
    5. Generate template briefing via `formatBriefing(sessionData, topEntities, state)`
    6. Return `{ briefing, updatedState: state }`

    **processStop(sessionData, state, options = {}):**
    Returns `{ synthesis: object, updatedState: object }`
    1. Per D-15: always deliberation. Set `state.processing.deliberation_pending = true`, `state.processing.last_deliberation_id = crypto.randomUUID()`, `state.processing.deliberation_type = 'rem_synthesis'`, `recordSpawn(state)`
    2. Update `state.self_model.recent_performance` from `state.injection_history`
    3. Generate template synthesis via `formatSynthesis(sessionData, state)`
    4. Return `{ synthesis, updatedState: state }`

    **processPreCompact(compactData, state, options = {}):**
    Returns `{ summary: string, updatedState: object }`
    1. Generate compact summary via `formatPreCompact(state)`
    2. Prune entities with activation < 0.1 from `state.activation_map`
    3. Return `{ summary, updatedState: state }`

    **processPostToolUse(toolData, state, options = {}):**
    Returns `{ updatedState: object }`
    1. Extract entities from `toolData.tool_input.file_path || ''` combined with `toolData.tool_name || ''`
    2. Update activation map
    3. Return `{ updatedState: state }` (no injection -- PostToolUse is lightweight)

    **consumeDeliberationResult(state, options = {}):**
    Returns `{ injection: string, type: string } | null`
    1. Check if result file exists at `options.resultPath || DELIBERATION_RESULT_PATH`
    2. If not exists, return null
    3. Atomic consume: `fs.renameSync(resultPath, resultPath + '.consumed')` (prevents double-consumption per Pitfall 3)
    4. Read consumed file, parse JSON
    5. Clean up: `fs.unlinkSync(consumedPath)`
    6. Validate TTL: `Date.now() - new Date(result.timestamp).getTime() > (result.ttl_seconds || DELIBERATION_TTL_SECONDS) * 1000` -> discard if stale
    7. Validate correlation: `result.correlation_id !== state.processing.last_deliberation_id` -> discard if mismatch
    8. Return `{ injection: result.injection, type: result.type || 'deliberation' }`

    **writeDeliberationResult(result, state, options = {}):**
    Writes deliberation result file for state bridge
    1. Build result object: `{ status: 'complete', correlation_id: state.processing.last_deliberation_id, timestamp: new Date().toISOString(), injection: result.injection || '', agent_id: result.agent_id || 'inner-voice', ttl_seconds: DELIBERATION_TTL_SECONDS, type: result.type || 'deliberation' }`
    2. Write to `options.resultPath || DELIBERATION_RESULT_PATH` via `fs.writeFileSync`

    Module exports: `{ processUserPrompt, processSessionStart, processStop, processPreCompact, processPostToolUse, consumeDeliberationResult, writeDeliberationResult, DELIBERATION_RESULT_PATH }`
