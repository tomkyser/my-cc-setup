**curation.cjs** (~100-120 LOC):
Port from Python graphiti-helper.py curate_results() (lines 144-186), summarize_text() (~line 200), generate_session_name() (~line 220). Exports:

1. `callHaiku(promptName, variables, options = {})` -- shared low-level function:
   - Read OPENROUTER_API_KEY from `process.env`
   - If no API key: return `{ text: variables.fallback || '', uncurated: true }`
   - Load prompt via `loadPrompt(promptName)` from core.cjs
   - If no prompt: return `{ text: variables.fallback || '', uncurated: true }`
   - Interpolate variables into prompt.user using `prompt.user.replace(new RegExp('\\{' + key + '\\}', 'g'), value)` for each variable
   - POST to `config.curation.api_url` with model `config.curation.model`, messages array (system + interpolated user), max_tokens from `options.maxTokens || 500`, temperature 0.3
   - Use `fetchWithTimeout()` with `options.timeout || config.timeouts.curation`
   - Headers: `Authorization: Bearer ${apiKey}`, `Content-Type: application/json`
   - On success: return `{ text: data.choices[0].message.content, uncurated: false }`
   - On any failure (HTTP error, timeout, parse error): `logError(options.hookName || 'curation', ...)` then return `{ text: variables.fallback || '', uncurated: true }`

2. `curateResults(memories, contextText, options = {})` -- curation wrapper:
   - If no API key or empty memories: return `memories ? '[uncurated]\n' + memories.slice(0, 500) : ''`
   - Call `callHaiku('curation', { memories, project_name: options.projectName || 'unknown', session_type: options.sessionType || 'startup', prompt: contextText || '' }, { ...options, fallback: '[uncurated]\n' + (memories || '').slice(0, 500) })`
   - Return result.text

3. `summarizeText(text, options = {})` -- summarization wrapper:
   - Call `callHaiku('session-summary', { content: text }, { ...options, maxTokens: 300, timeout: config.timeouts.summarization, fallback: text.slice(0, 200) })`
   - Return result.text

4. `generateSessionName(summaryText, options = {})` -- session naming wrapper:
   - Call `callHaiku('session-name', { summary: summaryText }, { ...options, maxTokens: 50, fallback: '' })`
   - Return result.text.trim() -- clean whitespace/quotes from name

**episodes.cjs** (~30-40 LOC):
Port from Python graphiti-helper.py cmd_add_episode() and _extract_content(). Exports:

1. `extractContent(response)` -- parse MCP JSON-RPC response:
   - If `!response || response.error` return `''`
   - `const content = (response.result || {}).content || []`
   - Filter items where `item.type === 'text' && item.text`, join with '\n'

2. `addEpisode(content, groupId, options = {})` -- write episode to Graphiti:
   - Create MCPClient instance: `new MCPClient(options)`
   - Call `client.callTool('add_memory', { content, group_id: groupId })`
   - Log errors via `logError(options.hookName || 'episodes', ...)`
   - Return response (or null on error)
   - Always use try/catch, never bare .catch

**search.cjs** (~40-50 LOC):
Port from Python graphiti-helper.py cmd_search(). Exports:

1. `searchFacts(query, groupId, options = {})` -- search Graphiti facts:
   - Create MCPClient: `new MCPClient(options)`
   - Call `client.callTool('search_memory_facts', { query, group_id: groupId, max_facts: options.maxFacts || 10 })`
   - Return `extractContent(response)` (import from episodes.cjs)

2. `searchNodes(query, groupId, options = {})` -- search Graphiti nodes:
   - Create MCPClient: `new MCPClient(options)`
   - Call `client.callTool('search_nodes', { query, group_id: groupId, max_nodes: options.maxNodes || 5 })`
   - Return `extractContent(response)`

3. `combinedSearch(query, groupId, options = {})` -- facts + nodes:
   - Run `searchFacts` and `searchNodes` in parallel with `Promise.all`
   - Combine: `'## Facts\n' + facts + '\n\n## Entities\n' + nodes`
   - Return combined text (or empty string if both empty)

All modules: use `path.join(__dirname, '..', 'core.cjs')` for core.cjs requires. Use `path.join(__dirname, 'mcp-client.cjs')` for mcp-client requires. No bare `.catch(() => {})` -- always log via logError.
