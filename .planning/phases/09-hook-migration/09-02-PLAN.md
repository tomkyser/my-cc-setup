**sessions.cjs** (~180-220 LOC):
First line: `// Dynamo > Ledger > sessions.cjs`

Requires: `path`, `fs`, `os`, and `logError` from core.cjs via `path.join(__dirname, '..', 'core.cjs')`.

Constants:
```javascript
const SESSIONS_FILE = path.join(os.homedir(), '.claude', 'graphiti', 'sessions.json');
```

Functions to implement:

1. `loadSessions(filePath)` -- filePath defaults to SESSIONS_FILE:
   - try: `JSON.parse(fs.readFileSync(filePath, 'utf8'))`
   - If result is not an array, return `[]`
   - catch: return `[]`

2. `saveSessions(sessions, filePath)` -- filePath defaults to SESSIONS_FILE:
   - `fs.mkdirSync(path.dirname(filePath), { recursive: true })`
   - Write to `filePath + '.tmp'`: `JSON.stringify(sessions, null, 2) + '\n'`
   - `fs.renameSync(tmp, filePath)` -- atomic rename

3. `indexSession(timestamp, project, label, labeledBy, options = {})`:
   - filePath from options.filePath or SESSIONS_FILE
   - Load sessions
   - Find existing by timestamp match
   - If existing and `existing.labeled_by === 'user'`: return (never overwrite user labels)
   - If existing and `!label && existing.label`: return (don't replace with empty)
   - If existing: update label, labeled_by; if options.namedPhase, set named_phase; if project non-empty and not 'unknown', update project
   - If new: push `{ timestamp, project, label, labeled_by: labeledBy }` plus named_phase if provided
   - Save sessions

4. `listSessions(options = {})`:
   - Load sessions from options.filePath or SESSIONS_FILE
   - If options.project: filter by project match
   - Sort by timestamp descending (localeCompare reversed)
   - If options.limit: slice to limit
   - Return filtered/sorted array

5. `viewSession(timestamp, options = {})`:
   - Load sessions, find by timestamp, return entry or null

6. `labelSession(timestamp, label, options = {})`:
   - Load sessions, find by timestamp
   - If not found: return false
   - Set entry.label = label, entry.labeled_by = 'user'
   - Save sessions, return true

7. `backfillSessions(nameGenerator, options = {})`:
   - Load sessions
   - Filter entries where `!entry.label && entry.labeled_by !== 'user'`
   - For each: `entry.label = await nameGenerator(entry)`, `entry.labeled_by = 'auto'`
   - Save sessions
   - Return count of backfilled entries

8. `generateAndApplyName(timestamp, project, nameGenerator, namedPhase, options = {})`:
   - Call `const name = await nameGenerator()`
   - If name: `indexSession(timestamp, project, name, 'auto', { ...options, namedPhase })`
   - Return name or ''

Export: `{ SESSIONS_FILE, loadSessions, saveSessions, indexSession, listSessions, viewSession, labelSession, backfillSessions, generateAndApplyName }`
